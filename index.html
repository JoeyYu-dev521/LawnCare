本版本的产品原则

  • 不要让用户在 7 个标签里迷路，首页只保留 4 个大动作。
  • 只展示今天能稳定工作的功能，避免让用户误以为拍照会自动判断。
  • 先把 HOA 证据链做好：日期、操作、产品、照片、状态评分。
  • 建议必须可解释，方便家人或 HOA 理解。

Today

今天的重点任务

今天只需要记住 3 件事

  1. 先观察：颜色是否均匀、有无圆形黄斑、有无爬地杂草。
  2. 再操作：割草、浇水、施肥、除草不要全部挤在同一天。
  3. 最后记录：拍一张照片，保存到 HOA 记录。

Manual Problem Guide

手动草坪问题判断

使用方法：先选最像的症状，再对照“怎么看”和“下一步怎么做”。如果不确定，保存照片到 HOA 记录，连续 1–2 周观察变化。

拍照记录标准

  • 每次从同一个方向拍：门口、车道边、邮箱旁等固定位置。
  • 近照 + 远照各一张：近照看杂草，远照看整体进度。
  • 照片当天同时记录天气、操作和产品名称。

Shopping Guide

产品指南

购买和使用任何除草剂、杀菌剂或肥料前，请以产品标签为准，确认适用于 Bermuda grass，并遵守当地规定。

施肥

用途：生长季维持草坪颜色和密度。

例子：Scotts Turf Builder Bermuda / Southern Lawn 类产品。

提醒：九月后不要继续高氮施肥。

阔叶杂草

用途：Virginia Buttonweed、Clover 等阔叶杂草。

例子：Bermuda-safe broadleaf weed killer，或 Celsius WG 类专业产品。

提醒：高温喷药要谨慎，严格看标签。

莎草 / Nutsedge

用途：黄绿色、长得比草快的三角茎杂草。

例子：Nutsedge-specific 产品，如 SedgeHammer / Image 类。

提醒：不要手拔，容易刺激地下块茎扩散。

霉菌 / Brown Patch

用途：湿热季节出现圆形黄斑或棕斑。

例子：Scotts DiseaseEx 或 BioAdvanced Fungus Control 类。

提醒：配合早晨浇水，避免傍晚浇水。
